Field Types

FormForge supports 13 field types for capturing different kinds of data.

Text

Single-line short text input

Email

Email address with format validation

Number

Numeric input with optional min/max range

Rich Text

Multi-line paragraph textarea

Single Select

Radio button choices (one answer)

Multi Select

Checkbox group (multiple answers)

Date

Native date picker with optional min/max dates

Time

Native time picker

Rating

Star rating (configurable 1–10 stars, half-star option)

Signature

Canvas drawing pad for mouse/touch signatures, saved as image

User Picker

Search and select Confluence users. Single or multi-user mode. Shows resolved name or unresolved ID. Requires authentication.

Group Picker

Search and select Confluence groups. Single or multi-group mode. Requires authentication.

Link / URL

URL input with domain whitelist validation, HTTPS enforcement, and custom error messages.
