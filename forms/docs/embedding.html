Embedding Forms

Embed FormForge forms directly into Confluence pages or share them via direct links.

Embedding in Confluence

To embed a form in a Confluence page:

  1. Type /form in the page editor
  2. Select the FormForge macro from the search results
  3. Click the pencil icon to open the macro configuration
  4. Choose your form from the dropdown list

Direct Shareable Links

Share forms outside of Confluence pages using direct links. Open the Share modal to access three tabs: Link (direct URL), Embed (iframe code), and ID (form identifier for API use).
