Why Macro Toolkit?

Secure by Default

Runs on Forge — no external servers for most macros. Data stays in your Atlassian instance.

Admin Controls

Toggle individual macros on/off site-wide. External service macros disabled by default.

Client-Side Rendering

Most macros render entirely in the browser. No server round-trips on page view.

Theme Aware

Uses Atlassian Design System CSS variables. Adapts to light and dark themes.

Live Preview Editors

Fullscreen config editors with real-time preview. See changes as you type.

Collaborative
